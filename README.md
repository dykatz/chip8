CHIP8(6) - Games Manual

# NAME

**chip8** - a simple CHIP-8 system emulator

# SYNOPSIS

**chip8**
*game*

# DESCRIPTION

The game
**chip8**
plays a CHIP-8 ROM. It will appear in its own window.
